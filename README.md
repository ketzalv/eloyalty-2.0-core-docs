# Master Product Repo

## Propósito
Repositorio maestro de documentación del producto, enfocado en arquitectura, UX, negocio, personalización por marca y contexto para IA.

## Objetivos
- Centralizar la documentación del producto
- Separar core vs personalizaciones por marca
- Facilitar trabajo con IA
- Generar entregables para cliente y equipo interno

## Estructura general
- /docs → conocimiento base del producto
- /brands → información específica por marca
- /templates → plantillas reutilizables
- /assets → imágenes, diagramas y mockups
- /meetings → notas de sesiones y hallazgos

## Cómo usar este repo
1. Documentar primero el core del producto
2. Crear carpeta de marca a partir de /brands/brand-template
3. Registrar decisiones importantes en /docs/05-decisions
4. Mantener los entregables al cliente dentro de cada marca
5. Usar /docs/08-ai-context para dar contexto consistente a herramientas de IA
